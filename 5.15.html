A while (loop) statement is aways followed by a condition which, after being evaluated, is determined to be either true or false. If the condition is true, and is further followed by some other type of action or logic, then that action gets executed. Otherwise, if a while loop's condition evaluates to false, no looping occurs and instead, the code flows directly to the next statement following the while loop (if another statement exists).

At line 4, the condition thats defined after the first while loop statement ((i = i + 2) < 7) executes, is initially true. Note that the execution of this statement simultaneously assigns the variable i, the value 2. Therefore, the first iteration of the while loop statement, as a result of the increment operator - combined with the print statement at line 5, prints the output 3.

When the code flow returns to execute the second iteration of the while loop statement, the value that gets simultaneously assigned to the variable i (at line 4), this time, is 5. Similarly, the increment operator again combines with the print statement at line 5, but this time, prints the output 6.

The value 6, when applied to the condition that follows the while loop statement after the third loop iteration, returns false - and causes the flow of the code to stop/terminate.
