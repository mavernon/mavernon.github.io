The value 50 is assigned (at line 5) to replace the value of the first element (10), that occurs in the first array of the two-dimensional array that was created at line 3. This modified value of the first element, is mirrored in the reference to that same array ( int [ ] b) which was created by the assignment at line 4.

By using the array reference as an argument, the output to the println statement at line 6 confirms that the value assigned at line 5, simultaneously modifies both the first element of the array reference, and the first element of the original multi-dimensional array (created at line 3).
