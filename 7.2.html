This example demonstrates the use of both class inheritance and constructor chaining.

Constructor chaining forces class Y' default constructor at line 16 to reference the default constructor of the base class (line 6), before it executes - itself. This results in the base class default constructor at line 6 being invoked immediately after class Y is instantiated (line 13). When executed, the default base class constructor calls the one-argument base class constructor at line 3 (via the this keyword at line 7) - passing it the value (3) of the static variable "y" (located on line 2) as a argument.

Although the naming convention (i.e. x) used in assigning the parameter variable identifier at line 3 is somewhat confusing, it is legal and has no quantifiable impact on the outcome of the solution. Ultimately, the constructor executes and assigns the value of 12 to the static variable "y" (at line 4).

Following the assignment at line 4, the code flow returns to the constructor at line 6, and executes the statement at line 8 (increasing the value of the static variable "y" to 13). The flow then returns to the constructor at line 16, where the static variable is again increased by the value of 1, and ultimately output (as 14) at line 14.
