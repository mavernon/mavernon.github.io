The statement at line 4 creates a three-dimensional array reference named, multiarray.

The index location of an element in an array, is represented by an integer value, inside a pair of one - or more square brackets, that follow the name of the array The println statement argument at line 8, for example, represents the index location of an array element in multiarray.

To point to the location of an element in a one-dimensional array, a single pair of brackets that contain an integer value, would be used (e.g. myarray [4] ). Similarly, to point to the location of an element in a two-dimensional array, two pair of brackets - each containing an integer value, is used (e.g. myarray [4][1] ). Accordingly, specifying the location of an element, of a three-dimensional array, requires that three pair of brackets (each containing an integer value) be used.

Thus, the println statement argument at line 8, can be interpreted as "... output the array element located at index position [1][2][0], of the three-dimensional array, name multiarray". The value of the array element located at that index position, of the three-dimensional array, is 1.

Note that, the value of the integer inside of the pair of brackets (that represent an array element's index location) could be represented as the output of an expression - that calculates a value (e.g. myarray [4 % 4] ).
