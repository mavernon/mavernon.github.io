A while (loop) statement is aways followed by a condition which, after being evaluated, is determined to be either true or false. If the condition is true, and is further followed by some other type of action or logic, then that action gets executed. Otherwise, if a while loop's condition evaluates to false, no looping occurs and instead, the code flows directly to the next statement following the while loop.

In this example, the condition defined in the first while loop (at line 4) is initially true. This causes the flow of the code to continuously repeat, and increment both the value of "c1" and "i" (due to the ++ operator), until the loop's condition statement evaluates to false. Ultimately, the first while loop results in a value of "4" for variable "c1" - while the assignment statement (at line 8) that immediately follows it re-assigns the variable "i" to its initial value of zero, before the do while loop (at line 9) begins executing.

The purpose of a do statement is to execute a segment of code - at least once, before a while loop begins. This keeps any condition that follows a while loop, from restricting or otherwise impacting the do segment of code - for at least one iteration, before the do segment of code invariably becomes subject to the result of the while loop's condition statement.

This continues repeatedly, until (or unless) the while loop condition evaluates to false, at which time looping stops (interestingly however, the do segment of code always executes one final time - even after the while loop condition evaluates to false ...).
