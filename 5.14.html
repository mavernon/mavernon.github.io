The code logic thats included in the first while loop statement, at line 4, contains the short-circuit operator, &&. A short-circuit operator will not evaluate the second part of a statement, if the first part of the statement is false (as in this case). As a result, the first while loop statement (lines 4 and 5) simply fail to execute.

The second while loop statement, at line 6 modifies the value of the variable i (via a post-increment operator) after the statement is evaluated – repeatedly, until the value of i is such that the while loop statement returns the value false and fails to execute. At this point, the code flow then immediately moves to the statement at line 7 and executes; outputting the last value that was incremented by the post-increment operator – 14, before the entire program terminates.
