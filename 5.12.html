The int type designation, at the beginning of the third line, is syntactical short-hand that eliminates the need to insert the int type before every variable declaration that falls within the same statement (e.g. counter).

The while loop statement (at line 4) evaluates to true – ten times, repeatedly. However, the logic thats associated with the if statement, inside of the the while loop , is true only every other time that the while loop statement repeats. The result is that, the variable counter increases by one, five times – before the while loop (that contains it) evaluates to false.

When the while loop (at line 4) finally does evaluate to false, the code flow immediately jumps to the statement at line 7, and executes it – before the program ends; resulting in the output 5.
